# Donna Website

This repository contains the Leaf source for [donna-lang.org](https://donna-lang.org).

The site is generated from Markdown content, theme templates, and static assets, then deployed to GitHub Pages.

## Build

```sh
leaf build
```

The generated site is written to `public/`.
